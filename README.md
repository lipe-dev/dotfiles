# dotfiles

Minimal configs for Mac, Windows, and Linux (WSL Ubuntu).

Clone the repo anywhere, then run the install script for your platform from the repo root.

## Linux (WSL Ubuntu)

```bash
bash linux/install.sh
```

## Mac

```bash
bash mac/install.sh
```

## Windows

Run PowerShell as Administrator:

```powershell
.\windows\install.ps1
```
